Como Funciona

Este site faz parte de um pequeno projeto requerido pelo FreeCodeCamp, nenhum serviço é desponibilizado pelo mesmo. A prática de HTML e CSS tiveram foco na construção desta página. As imagens inseridas aqui e os vídeos são meramente para fins demonstrativos. Abaixo segue alguns dos conteúdos que foram praticados:

  • Displays: Grid, Flex;
  • Positions: Fixed, Absolute;
  • Animations.
